# Seishinkan Aikido Kuching

The official website for Seishinkan Aikido Dojo in Kuching, Sarawak. The site
shares the dojo's history, affiliation, class schedule, fees, location, and
WhatsApp contact details.

## Prerequisites

- Node.js `>=22.13.0`

## Quick Start

```bash
npm install
npm run dev
npm run build
```

This starter does not use `wrangler.jsonc`.

## Commands

- `npm run dev`: start local development
- `npm run build`: verify the vinext build output
- `npm test`: build and run the rendered HTML checks

Built with Next.js-compatible React through [vinext](https://github.com/cloudflare/vinext)
and deployed on Cloudflare Workers through OpenAI Sites.
